Quiz Questions :

map() : map() method creates a new array from calling a function for every array element.

filter() : filter() method returns a new array with all elements that pass the test defined by the given function.

find() : find() method returns the value of the first array element that satisfies the provided test function.

forEach() : forEach() method calls a function for each element in an array. It is not executed for empty elements.

Local Storage : Local storage is a web storage API that allows web applications to store data locally within a user's web browser. The data is persistent and will remain even after the user closes the browser window or restarts the computer

Session Storage : Session storage is a web storage API that allows web applications to store data locally within a user's web browser for the duration of a single browsing session. The data stored in session storage is only accessible within the same browser tab or window and is lost when the tab or window is closed or the browser is shut down.

Cookies : Cookies are small text files that are stored in a user's web browser by websites. They are typically used to store user preferences, login information, and other data that needs to persist across different sessions.

Event Loop : The event loop is a fundamental concept in JavaScript that allows for asynchronous programming. And event loop monitors the call stack and task queue in JavaScript, executing tasks in a non-blocking way to handle asynchronous tasks efficiently while maintaining a responsive user interface.
